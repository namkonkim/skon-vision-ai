  2-2. Pooling : Convolution이 이미지의 특징을 추출하는데 3*3 필터 크기로 가정하고.. 이미지의 더 넓은 영역으로 특징을 추출을 해 나가야 하는데, Convolution Filter 크기를 늘리면 연산량이 기하급수적으로 늘어나고 학습에 나쁜영향을 미치기 때문에, Convolution Filter 크기를 유지한 채 이미지의 주요 정보를 압축해 나가는 과정이 Pooling이다. Max pooling, Avg pooling 에 대해 설명하고 비교 및 장/단점을 소개해줘.

다음으로 "chapter 12. 성능을 높히는 라벨링 가이드라인" 교육 내용을 작성하자.
이 Chapter에 대해서는 Vision AI Task별로 이미지 5장씩을 직접 라벨링을 해보고, 성능을 높히기 위해 어떻게 라벨링을 하면 좋을 지 간단하게 가이드를 하고, 오토라벨링 컨셉에 대해 설명을 하려고 해.
Task는 Classification (이미지 폴더로 구분), Object Detection (BBOX), Segmentation (Polygon), Oriented Boundary Box (BBOX with angles) 를 해볼거고, 라벨링툴로는 Labelme 를 설치 후 수행하도록 가이드 해줘.
그리고 라벨링할 이미지는 인터넷에 찾거나 너가 직접 생성을 해줘도 돼. 단, Annotation 을 하는데 경계가 뚜렷하거나 Box치기 쉬운 이미지도 준비를 하지만 경계가 흐릿하고 Box치기 어려운 이미지 샘플도 준비를 해서, 각 Task 별로 5장의 이미지를 준비해줘.

다음으로 성능을 높히기 위한 라벨링 가이드에 대해서는 너가 알아서 작성을 해줘. 라벨링 하는 사람들끼리 일관된 기준을 잡아야 되는데, 어떻게 기준을 잡으면 좋은 지도 포함해서.

마지막으로 오토라벨링의 경우 SAM 모델을 사용하는 방법과 라벨링한 이미지를 가지고 간단한 모델을 직접학습시킨 후, 라벨링 안된 이미지를 추론하고 confidence가 높은 것들에 대해 일괄 라벨링 붙히는 방법에 대해 소개를 해주면 좋겠어.
